## 스터디 규칙
### 대면 스터디 전
1. 정해진 분량의 내용을 자신의 블로그에 정리한다.
2. 정리한 링크를 PR에 올린다.
3. 각자 무조건 질문을 issue탭에 1개씩 올린다.
4. 스터디원이 올린 질문에 대해 답을 한다.
   
### 대면 스터디
1. 매주 수요일 저녁 대면 스터디를 진행한다.
2. 이슈를 중심으로 토의한다.
